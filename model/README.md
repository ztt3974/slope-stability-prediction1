# 此目录用于存放训练好的模型文件

## 必需文件
- `ipso_bp_ensemble_model.pkl` - 训练好的集成学习模型

## 如何获取模型文件？

### 方法1: 从本地复制（推荐）
1. 在本地运行训练脚本生成模型：
   ```bash
   python ipso_bp_slope_stability_fixed.py
   ```
2. 将生成的 `.pkl` 文件复制到此目录

### 方法2: 联系开发者
- 如果你是项目协作者，可以向项目负责人索取模型文件

### 注意事项
- 模型文件大小通常在 50-200MB 之间
- 确保使用与训练时相同版本的 Python 和依赖包
- 如果遇到编码问题，请使用修复版训练脚本重新训练
